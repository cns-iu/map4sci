count_crossings¶

count_crossings(
    G, edges_to_compare=None, stop_when_found=False, ignore_label_edge_cr=False
)

Source: src/quality_measurement/original/crossings.py

Counts the crossings of the given graph G< t>. The crossing count can be executed only on the given list of edges of G< t> passed as input in edges_to_compare< t>. Also the execution can stop as soon as a crossing is found if the boolean value stop_when_found< t> is set to True. If the vertices have labels with given height and width the crossings that occur below the labels can be ignored if the boolean value ignore_label_edge_cr< t> is set to True. Return a list of crossings where each element has the crossing edge and the intersection point.
